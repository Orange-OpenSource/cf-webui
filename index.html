CF-WebUI

Single page Cloud Foundry Web User Interface using AngularJS and Bootstrap

View project on GitHub

Overview

Cloud Foundry is the OpenSource Platform as a Service (PaaS) Framework on which many PaaS offerings are based (e.g. Pivotal Web Services, HP Helion, IBM BlueMix, Swisscom Application Cloud, etc.). It allows the developers to provide, manage and scale their application in the cloud very easily and quickly.

For end-users Cloud Foundry provides a REST based API and a command line tool (CLI). So far no up-to-date modern and free open source web front-end is available. Commercial providers have developed their own proprietary user interface.

The aim of this project is to design and develop an open source modern interactive web user interface for Cloud Foundry.

Getting started

  1. Clone the project: git clone https://github.com/icclab/cf-webui.git
  2. Change to the cf-webui directory: cd cf-webui
  3. Change the endpoints to your desired Cloud Foundry instance in app/app.constant.js.
  4. Update the manifest.yml with your desired deployment options. e.g. (change host and domain-name)
  5. Push this application to Cloud Foundry with cf Command Line Interface (CLI): cf push.
  6. Enjoy using CF WebUI!

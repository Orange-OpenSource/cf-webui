CF-WebUI

Single page Cloud Foundry Web User Interface using AngularJS and Bootstrap

View project on GitHub

Overview

Cloud Foundry is the OpenSource Platform as a Service (PaaS) Framework in which the most recent public PaaS offerings are based (eg. HP Helion, IBM BlueMix, Swisscom Application Cloud, etc.). It allows the developers to provide, manage and scale their application in the cloud very easily and quickly.

For end users Cloud Foundry provides a REST based API and a command line tool. So far no up to date modern and free Open Source Web-Frontend is available. The commercial providers have developed their own UI. The aim of this work is to design, develop and provide a modern interactive Web Frontend for Cloud Foundry, an open source tool available to the community.

Getting started

  1. Clone the project: git clone https://github.com/chessjs/chess
  2. Change directory to cf-webUI: cd cf-webUI
  3. Change the endpoints to your desired Cloud Foundry instance in /app/app.constant.js.
  4. Change the manifest.yml to your desired options. E.g.:
    ---applications:
       - name: cf-webui
        memory: 128M
        host: console
        domain: pws.icclab.io
  5. Push this application to Cloud Foundry with cf Command Line Interface (CLI): cf push.
  6. Enjoy the CF WebUI!
